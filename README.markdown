# random_text

http://github.com/toy/random_text/tree/master

## DESCRIPTION:

Class to make usage of http://vesna.yandex.ru/ and http://lipsum.com/ easier

## SYNOPSIS:

    include RandomText

    Lorem.word
    Lorem.words(5)

    Lorem.uniq_words(10)

    Lorem.sentence
    Lorem.sentences(5)

    Lorem.paragraph
    Lorem.paragraphs(5)

Same can be done for Vesna

    Vesna.word

    lorem word
    lorem 10 words
    lorem 20 uniq_words
    lorem 7 sentensec
    lorem 3 paragraphs
